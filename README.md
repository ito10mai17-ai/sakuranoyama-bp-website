## 公式LINEの統合

公式LINEの友だち追加リンクを、ヘッダー・各CTA・スマホ固定ボタンへ反映済みです。

```text
https://lin.ee/RKEljXL
```

QRコード表示には、以下の公式QR画像を使用しています。

```text
https://qr-official.line.me/gs/M_603zjtdu_BW.png?oat_content=qr
```

## 開くときの注意

レイアウトが崩れる場合は、`index.html` だけを単体で開いている可能性があります。

このサイトは以下のフォルダ構成が必要です。

- index.html
- styles.css
- script.js
- assets/images/site-logo.png
- assets/images/hero-student-sakura.jpg

必ずZIPを解凍してから、解凍後のフォルダ内にある `index.html` を開いてください。
